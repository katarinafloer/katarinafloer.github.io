# Kate Floer Personal Website

A small static personal website for GitHub Pages.

## Edit Kate's List

Open `script.js` and add or replace items in the `savedThings` array:

```js
{
  title: "Example title",
  description: "A short note about why this is worth saving.",
  url: "https://example.com",
  tag: "Paper",
  date: "June 2026",
}
```

## Publish With GitHub Pages

In the GitHub repository settings, enable Pages and choose the `main` branch with the repository root as the source.
